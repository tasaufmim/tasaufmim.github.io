Projects

The Whisperer: A Hyper-Optimized Model for Flawless, Live-Stream Bengali Sign Language Decoding!

This research unveils The Whisperer, a hyper-optimized, low-latency deep learning architecture for real-time Bengali Sign Language (BdSL) decoding from live video feeds. The system leverages Spatio-Temporal Graph Convolutional Networks (STGCN) combined with LSTM networks to precisely track complex hand geometries and non-manual features. This robust approach ensures flawless, near-instantaneous translation, significantly advancing accessibility and setting a new technical benchmark for continuous sign language interpretation.

CaptionBot: The End-to-End Image Annotation and Dataset Preparation Engine

GitHub Repository

A robust, automated solution designed to dramatically accelerate the machine learning data preparation lifecycle by transforming raw images into meticulously captioned, training-ready datasets.

Domain-Specific Captioning: Achieved high-fidelity annotation by fine-tuning the Salesforce BLIP model specifically for domain-relevant image captioning, utilizing a proprietary, custom dataset to ensure unparalleled contextual accuracy.

Integrated ML Pipeline: The core is an efficient, end-to-end PyTorch pipeline meticulously built for seamless data preprocessing, comprehensive model training, and rigorous evaluation, leveraging the power and flexibility of Hugging Face Transformers.

High-Efficiency Deployment: Validation conducted in Google Colab, utilizing a powerful stack including PyTorch, Transformers, Pillow, and tqdm to ensure efficient resource utilization and reliable training metrics.

Full-Stack AI Chatbot with RAG

GitHub Repository

This project developed a highly scalable, full-stack AI Chatbot engineered for real-time text generation and enhanced contextual accuracy via Retrieval-Augmented Generation (RAG). The modular architecture features a Next.js frontend and an Express.js backend integrated with the Groq API for ultra-low-latency responses. Key implementations include dynamic in-message chat history, robust API error handling, and strategic frontend/backend separation, collectively providing a stable, fast, and knowledge-grounded user experience.
